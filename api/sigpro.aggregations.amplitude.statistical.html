sigpro.aggregations.amplitude.statistical.kurtosis(amplitude_values, fisher=True, bias=True)[source]¶

Compute the kurtosis ,Fisher or Pearson, of an array of values.

Parameters

  • amplitude_values (numpy.ndarray) – Array of floats representing signal values.

  • fisher (bool) – If True, Fisher’s definition is used (normal ==> 0.0). If False, Pearson’s definition is used (normal ==> 3.0). Defaults to True.

  • bias (bool) – If False, then the calculations are corrected for statistical bias. Defaults to True.

Returns

The kurtosis value of the input array. If all values are equal, return -3 for Fisher’s definition and 0 for Pearson’s definition.

Return type

float
