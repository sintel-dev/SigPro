class sigpro.basic_primitives.Kurtosis(fisher=True, bias=True)[source]¶

Bases: sigpro.primitive.AmplitudeAggregation

Kurtosis primitive class.

Computes the kurtosis value of the input array. If all values are equal, return -3 for Fisher’s definition and 0 for Pearson’s definition.

Parameters

  • fisher (bool) – If True, Fisher’s definition is used (normal ==> 0.0). If False, Pearson’s definition is used (normal ==> 3.0). Defaults to True.

  • bias (bool) – If False, then the calculations are corrected for statistical bias. Defaults to True.
